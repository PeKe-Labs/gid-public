# Roadmap

GID is built foundations-first: settle the irreversible decisions before drafting
anything normative.

- **Foundations** — the one-way doors: wire envelope, versioning, the extension
  mechanism, addressing, the conformance core. *Being settled now.*
- **Draft spec** — the normative sections ([`spec/`](spec)), published as each stabilizes.
- **Reference implementation** — the small `.h`/`.c` encoder/decoder ([`src/`](src)).
- **Catalog & adopters** — the standard vocabulary, grown with the first devices and apps.

Live roadmap: <https://gid.pekelabs.com/roadmap>
