# Contributing

GID is built in the open, and it gets better the more it's argued with.

Right now the work is in the **foundations** — the irreversible decisions the rest
of the standard rests on (see [`decisions/`](decisions)). The most valuable
contributions at this stage are sharp questions, real-world constraints, and
prior art we've missed — not code.

- Discuss — <https://discord.gg/hAAR933XY6>
- Contact — <info@pekelabs.com>

A fuller contributor guide follows as the spec and reference implementation open up.
